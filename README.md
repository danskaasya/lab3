# Лабораторна робота №3 - MVC Додаток для роботи з книгами

## Опис
Консольний додаток, реалізований з використанням архітектурного шаблону MVC, для роботи з масивом даних книг.

## Структура проекту
- `Book.java` - Модель книги з полями: назва, автор, видавництво, рік видання, кількість сторінок, ціна
- `BookController.java` - Контролер для обробки масиву даних книг
- `BookView.java` - View для відображення меню та результатів
- `BookPublisherComparator.java` - Comparator для сортування книг за видавництвами
- `Main.java` - Головний клас додатку

## Функціональність
1. **Отримання списку книг зазначеного автора**
2. **Отримання списку книг, які видані зазначеним видавництвом**
3. **Отримання списку книг, виданих пізніше вказаного року**
4. **Відсортування книг за видавництвами** (використовує Comparator)

## Запуск програми

### Варіант 1: Через IntelliJ IDEA (рекомендовано)
1. Відкрийте проект в IntelliJ IDEA
2. Відкрийте файл `src/Main.java`
3. Натисніть кнопку "Run" або використайте комбінацію клавіш `Shift+F10`

### Варіант 2: Через командний рядок
1. Встановіть JDK (Java Development Kit)
2. Відкрийте командний рядок в папці проекту
3. Запустіть `run.bat` або виконайте команди:
   ```
   cd src
   javac *.java
   java Main
   ```

## Меню програми
- **1** - Показати всі книги
- **2** - Знайти книги за автором
- **3** - Знайти книги за видавництвом
- **4** - Знайти книги після вказаного року
- **5** - Відсортувати книги за видавництвами
- **6** - Автоматичний пошук (випадкові дані)
- **0** - Вихід

## Особливості реалізації
- Використовує архітектурний шаблон MVC
- Містить 15 тестових книг з випадковими даними
- Підтримує автоматичний пошук з випадковими параметрами
- Виводить відповідні повідомлення, якщо книги не знайдено
- Відображає вихідний масив після створення
